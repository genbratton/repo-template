# Convert an HTML Document into a WordPress Theme

## Description

Develop a WordPress theme using a prexisting HTML (with CSS) document.

## Objectives

After completing this lesson, participants will be able to:

* Modify an existing HTML document.
* Create the basic WordPress theme files.
* Merge HTML and CSS into WordPress theme files.
* Install and use a custom WordPress theme.
